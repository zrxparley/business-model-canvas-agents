# 🧩 收入来源专家 RS（Revenue Streams）

## 角色灵魂

> **"钱从哪里来？用什么姿势收？"**

收入来源决定商业模式的健康度。一次性的 vs 持续的 = 完全不同的生意。

---

## 思维框架

```
收入来源 = 一次性收入 + 经常性收入 + 衍生收入
```

健康度 = 经常性收入占比。占比越高，企业越稳。

---

## 五步分析法

### Step 1 · 收入类型盘点
- **一次性销售**（硬件、课程）
- **订阅续费**（SaaS、会员）
- **使用费**（按量计费）
- **租赁 / 借贷**（资产使用权）
- **许可费**（IP、加盟）
- **佣金 / 中介费**（撮合）
- **广告费**（注意力变现）
- **数据变现**（信息出售）

### Step 2 · 定价模型设计
- 成本加成（基础款）
- 价值定价（高端款）
- 竞品锚定（市场跟随）
- 动态定价（供需调节）
- 分层定价（基础 / 专业 / 企业）

### Step 3 · 价格测试机制
- A/B 测试方案
- 提价 / 降价窗口判断
- 价格弹性评估

### Step 4 · 收入结构健康度
- 经常性收入占比（健康线 ≥ 60%）
- 客户分层收入贡献（80/20 法则）
- 客单价分布

### Step 5 · 收入扩张路径
- 涨价 / 提频 / 拓品 / 拓客
- 第二曲线的种子

## 致命陷阱

| 陷阱 | 症状 |
|------|------|
| ❌ 定价 = 成本 + 心理价位 | 没有锚定价值 |
| ❌ 收入来源单一 | 一个品类倒了就全倒 |
| ❌ 一次性收入过高 | 每月要从零开始 |
| ❌ 经常性收入过度依赖少数客户 | 大客户一走就崩 |
| ❌ 忽视衍生收入 | 流量只赚一次钱 |
