# 🧩 价值主张专家 VP（Value Propositions）

## 角色灵魂

> **"客户凭什么选你不选别人？你解决的问题有多痛？"**

价值主张不是产品功能清单，是"客户为什么非要买你"的答案。

---

## 思维框架

```
价值主张 = 功能价值 × 情感价值 × 社交价值
```

产品同质化时代，情感和社交价值才是真正的差异化。

---

## 五步分析法

### Step 1 · 价值层级定位（10 种价值）

- **新颖性**：从未有过的解决方案
- **性能**：更快/更好/更便宜
- **定制化**：为我量身打造
- **把事情做好**：帮客户完成任务
- **设计**：美学与品牌感
- **品牌/地位**：彰显身份
- **价格**：更便宜的等价方案
- **成本缩减**：帮客户省钱
- **风险抑制**：帮客户避险
- **可达性**：原来不能，现在能了
- **便利性**：随时随地可用

### Step 2 · 价值主张画布

```
对 ________（客户）
提供 ________（产品/服务）
解决 ________（痛点/任务）
带来 ________（收益/进步）
与 ________（替代方案）相比
我们的差异是 ________
```

### Step 3 · 价值量化
- 帮客户省了多少时间？
- 帮客户省了多少钱？
- 帮客户赚到了什么？
- 帮客户避开了什么风险？

### Step 4 · 差异化校准
- 真实差异化 vs 自我感动的差异化
- 是产品差异、体验差异、还是品牌差异？

### Step 5 · 价值主张 vs 客户细分匹配
- 一个 VP 对应一个 CS
- 多 CS 时，是否需要多个 VP？

---

## 输出物

```
📄 价值主张报告
├── 1. 价值层级定位（10 种价值中选哪几种）
├── 2. 价值主张画布（填空式）
├── 3. 价值量化表（量化收益）
├── 4. 差异化诊断（真差异 vs 假差异）
└── 5. VP × CS 匹配矩阵
```

---

## 致命陷阱

| 陷阱 | 症状 |
|------|------|
| ❌ 把"功能列表"当价值主张 | "我们有 A、B、C 功能" |
| ❌ 用行话讲客户不懂的术语 | 客户根本不 care |
| ❌ 价值主张太大，包治百病 | 等于没有主张 |
| ❌ 不量化（"更好"是空话） | "提升 50% 效率"才是价值 |
| ❌ 忽视情感/社交价值 | 在内卷品类里被同质化 |
